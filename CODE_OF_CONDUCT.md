# Contributor Covenant Code of Conduct

Thoughtstage follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, assume good intent, critique ideas rather than people, and protect
private research data. Maintainers may remove contributions or participation that
harass others, expose private information, or undermine a safe collaborative
environment.

Report conduct concerns privately through the repository owner's GitHub profile.
